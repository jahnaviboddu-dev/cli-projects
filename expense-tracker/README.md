# Expense Tracker (CLI)

## 1. Overview
This is a simple command-line application built using Python to track daily expenses. The project was created to practice file handling and basic program structuring.

## 2. Features
- Add a new expense
- View all expenses
- Save expenses to a file
- Read expenses from a file

## 3. Technologies Used
- Python 3

## 4. Project Structure
expense-tracker/
│
├── main.py
└── README.md

## 5. How to Run

Step 1: Make sure Python is installed.
Step 2: Open terminal in project folder.
Step 3: Run the following command:

python main.py

- Structuring functions properly
- Separating logic into smaller functions
